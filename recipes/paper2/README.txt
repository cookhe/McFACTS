Decoding the input filenames.

example: paper2_sg_t0p25.ini

Decimals are written as "p": 0p25 => 0.25

sg : Sirko & Goodman 2006 disk model
tqm : Thompson, Quataert, & Murray 2003 disk model
t : time
    disk lifetime in Myr
r : radius
    in gravitational radii
e : eccentricity
g : gamma (IMF index)
fr : retrograde fraction
s : spin
    width of Gaussian for initial spin draws
